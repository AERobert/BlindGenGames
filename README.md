# Web Games Vibecoded by LLMs to be playable with screen readers

These games were all made for me and for my personal taste in spoken interface. Mac OS Safari probably gives the best play experience. These games also have had pretty much no testing by non screen reader users (so they might very well look horrible). If you are not me and you have fun with any of these games, great! 

Report bugs or suggest more games on [the Github repo of this project](https://github.com/AERobert/BlindGenGames). All of the games are under the MIT license so feel free to use code in your projects.
